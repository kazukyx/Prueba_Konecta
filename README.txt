PRUEBA KONECTA

1- En la carpeta "DB" del proyecto "Prueba_Konecta" se encuenta el script de base de datos "konecta.sql" el cual se debe ejecutar primero para que pueda ver el visualizar el proyecto.
2- En la carpeta "config" del proyecto "Prueba_Konecta" se debe configurar la base de datos.
3- Abrir en el navegador que prefiera la raiz del protecto EJEMPLO "http://localhost/Prueba_Konecta/index.php"
4- Modulo Administrador: Encontrar un CRUD de productos, Producto mas vendido y Producto con mas stock.
5- Modulo Venta: Puede realizar las ventas de los productos registrados desde el Modulo Administrador.
